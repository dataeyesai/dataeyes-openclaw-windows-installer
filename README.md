# DataEyes OpenClaw Windows Installer

DataEyes OpenClaw 的 Windows 安装包分发仓库。

这个仓库只负责：
- 提供 Windows 一键安装包下载入口
- 发布版本说明

不建议把安装包直接提交进 Git 仓库历史，因为安装包体积较大。
